10a §3 — Key vs index join

Two charts of the same five items share one shuffle button. The only difference between them is the second argument to .data() — the left chart uses an index join (no key), the right uses a key join (d => d.id). Click Shuffle a few times and watch each circle's colour: with the index join the colour stays glued to its slot; with the key join the colour follows the letter. Source tutorial →
