10e §2 — Static scatterplot (baseline)

The starting point. Gapminder 2007 — GDP per capita (log) against life expectancy, circle area encodes population, colour encodes continent. Wheel and drag do nothing yet. The next three examples progressively add zoom behaviour to this chart. Source tutorial →

What this example shows

A complete, conventional scatterplot — the same shape we will keep using as we add zoom behaviour. The point is to fix the visual reference: every change in §3–§5 is a delta against this chart.

  • Four visual channels from one row of Gapminder: x = GDP per capita on a log scale, y = life expectancy (linear), circle area = population (via scaleSqrt), colour = continent (categorical palette).
  • Margin convention via makeCanvas: the inner plot group is translated by (margin.left, margin.top) so axes have room to render outside the data area.
  • Native SVG tooltips: each circle has a <title> child, which the browser renders on hover. No JS needed — useful as a baseline before custom tooltips (covered in 10c §3).
  • No zoom yet: wheel and drag intentionally do nothing. The next three examples each add one ingredient on top.
