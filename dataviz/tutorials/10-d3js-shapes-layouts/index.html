DV10 — Advanced D3

L9 finished with a complete static scatterplot (09e). DV10 turns that static foundation into a living chart by adding the five things a static chart can't do: change, time, input, composition, navigation — one tutorial each.

What you should already know

  • SVG primitives and the path mini-language (09a)
  • The static data join — selectAll · data · join("tag") (09b)
  • Scales, axes, and shape generators — scaleLinear, scaleBand, scaleLog, scaleSqrt, d3.line, d3.area (09c)
  • Loading CSVs and helpers (09d)
  • The first complete chart — Gapminder scatterplot (09e)

The five moves

  1. Change — what happens when the data updates (10a)
  2. Time — animate the change (10b)
  3. Input — let the reader trigger the change (10c)
  4. Composition — layouts that turn data into geometry (10d)
  5. Navigation — zoom and pan when the data is denser than the screen (10e)

Tutorials

Run them in order — each one builds on the previous.

  1. 10a — Update Pattern and selection.join
    What 09b's static .join does on the second call. The older explicit form (.exit() · .enter() · .merge()); modern .join with three callbacks; key functions vs index joins (interactive shuffle); sortable bar chart of countries by population.
  2. 10b — Transitions
    selection.transition() basics; transitions after a join; chaining; tuning with duration, delay, ease; staggered transitions via per-datum delay; transitions on enter / exit; bar chart race.
  3. 10c — Interactions
    selection.on for browser events; the event → state → render loop pattern; hover tooltips; click-to-toggle filters; year-slider scrubbing; full Gapminder interactive scatterplot; d3.brush for range filtering.
  4. 10d — Layouts
    Bar chart (the simplest layout-driven chart); scaleBand padding knobs; d3.stack layout (input → output → nested join); streamgraph (same stack with stackOffsetWiggle); a which layout for which question? decision table covering stack · pie · hierarchy · force · chord · sankey.
  5. 10e — Zoom and Pan
    d3.zoom in four visible stages: static scatterplot → naive group transform → transform.rescaleX / rescaleY on the scales → clip-path to the plot rectangle. When to use the rescale-scales pattern vs the group-transform pattern. Production checklist: responsive viewBox, accessible palettes, ARIA, when to switch to <canvas>.

Setup

The tutorials are plain HTML files and self-contained — every asset they need (_shared/chart.css, _shared/chart.js, datasets/gapminder.csv) is inside this folder. Serve them with a local HTTP server:

cd 10-d3js-shapes-layouts
python -m http.server 8000
# open http://localhost:8000/

D3 is loaded from the CDN (https://d3js.org/d3.v7.min.js); no installation required.

Per-demo files

The examples/ subfolder contains one self-contained HTML file per demo — each one ships its own copy of makeCanvas and styling, so it can be opened on its own (or copied into another project) without the rest of the tutorial folder. Useful for grabbing one pattern in isolation.
