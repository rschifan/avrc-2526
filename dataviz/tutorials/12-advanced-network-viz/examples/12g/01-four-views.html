12g — Same data, four views

The 30 best-connected Game-of-Thrones characters drawn four ways at once — force, matrix, chord, bundling — over the same nodes, edges, and community colours. Hover any node in any panel and the same character lights up in all four. Source tutorial →

What this example shows

  • One shared hover, four layouts. Each panel registers a sync handler; hovering anywhere calls broadcast(id), which highlights that character in every view at once.
  • Each view answers a different question — global shape (force), pair lookup (matrix), weight distribution (chord), group-level flow (bundling).
  • Same colour scale (community) across all panels, so a colour means the same group everywhere.

Same colour = same community. Hover any node, cell, arc, or leaf.
