12c — Sankey layout & node alignment

A small synthetic flow — character origin house → allegiance → fate — laid out by the d3-sankey plugin. Band width is proportional to the number of characters on that path. Switch the node alignment and watch the layers snap. Source tutorial →

What this example shows

  • The plugin computes the geometry. d3.sankey() mutates each node with x0/x1/y0/y1 and each link with a width; rendering is then two ordinary joins (rects + sankeyLinkHorizontal paths).
  • nodeAlign chooses which layer each node snaps to — sankeyLeft/Right/Justify/Center.
  • Flow is conserved: a node's height equals its total throughput, so the bands in equal the bands out. Hover a node for its in/out totals.
