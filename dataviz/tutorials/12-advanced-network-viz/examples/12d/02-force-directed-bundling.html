12d — Force-directed edge bundling

The same GoT network bundled without a hierarchy (Holten & van Wijk), via the d3.ForceEdgeBundling library: subdivide each edge, pin the endpoints, and let the control points of nearby parallel edges attract one another. Source tutorial →

What this example shows

  • No hierarchy needed. FDEB takes node positions (from a quick force pass) and an edge list, and returns a poly-line of subdivision points per edge.
  • Compatible edges merge. Edges that run roughly parallel and near each other bundle together; the result emphasises overall edge-flow patterns.
  • Runs once. The algorithm compares every pair of edges (O(E²)), so it takes a moment on the full edge set — hence the brief "bundling…" message.
