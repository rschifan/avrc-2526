11g — Capstone: a complete GoT network viewer

Everything from 11a–11e in one demo: data load, force layout, community colour, centrality size, drag, hover-to-highlight, click-to-pin, semantic zoom, plus a search box and a community filter. Read the source as a checklist of every pattern in this lecture.

The viewer

Try this:

  • Type a name in the search box (e.g. Tyrion) — matching nodes light up red.
  • Click a community swatch in the legend — only that community is shown; click again to restore.
  • Hover a node — its neighbourhood stays opaque, the rest fades.
  • Shift+click a node — focus mode: it slides left and neighbours stack on the right. Click the empty background to relax back.
  • Click a node — it pins (orange ring); click again to release.
  • Drag any node anywhere; wheel + drag the background to zoom and pan with zoom-invariant sizing.
  • Toggle the community-hull overlay on/off.

What's where in the code

  • Layout — d3.forceSimulation with the four standard forces (11b).
  • Encoding — scaleOrdinal(schemeTableau10) for community, scaleSqrt for radius (11d).
  • Drag — makeDrag(sim) (11c).
  • Hover — precomputed neighbour set, opacity fade (11c).
  • Click pin — toggle fx/fy (11c).
  • Zoom — d3.zoom on a wrapper <g>, with semantic stroke / label-size adjustments (11e).
  • Search — input event applies a "match" highlight class.
  • Community filter — clicking a legend swatch toggles a community in/out of an active set; node and link display driven by membership.
